Instalação

Como ligar ao backend

Publique o backend como Web App e cole a URL em frontend/config.js.

1. Base de dados

Crie a base de dados do sistema e abra Extensões → Apps Script.

2. Apps Script

Cole o conteúdo de backend/Code.gs, guarde e faça Deploy → Web app.

3. Vercel

Suba a pasta frontend para GitHub/Vercel. Depois actualize APPS_SCRIPT_URL.

4. Membros

Use o cadastro Membros com colunas name, label, Telefone e os demais campos do cadastro.
